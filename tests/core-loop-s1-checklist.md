# Test Checklist — Core Loop S1 首次到店闭环

## A. 主路径（Happy Path）
- [ ] A1 Home 点击“拍照或选择照片”进入 Scanner
- [ ] A2 Scanner 选 1-5 张后点击“确认并分析”进入 Chat
- [ ] A3 Chat 出现“识别中/分析中”状态反馈
- [ ] A4 Chat 返回 >=2 条推荐，且每条有理由
- [ ] A5 添加推荐到 OrderCard，数量可增减
- [ ] A6 进入 WaiterMode 并展示原文菜名
- [ ] A7 返回 Chat 后订单与上下文仍在

## B. 异常恢复
- [ ] B1 识别失败时出现明确恢复动作（重试/补拍）
- [ ] B2 对话失败时可继续聊天或重试
- [ ] B3 网络慢时界面不假死，有进行中反馈

## C. 验证标准
- [ ] C1 单次流程 <= 3 分钟
- [ ] C2 无阻断性状态断裂
